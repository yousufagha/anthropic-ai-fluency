# anthropic-ai-fluency

Personal notes from Anthropic's AI Fluency: Framework & Foundations course, covering the four core skills for working effectively with AI systems.

[Delegation](delegation.md): deciding what to hand off to AI, and how much oversight it needs.

[Description](description.md): communicating clearly enough to get useful output.

[Discernment](discernment.md): evaluating AI output critically before trusting it.

[Diligence](diligence.md): following through and staying accountable for AI-assisted work.

Each note captures the core idea of the skill, a few key points from the course, and a practical takeaway I'm applying going forward.
